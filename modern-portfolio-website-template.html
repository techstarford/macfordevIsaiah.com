Project Description

Developed a responsive portfolio website for a photographer, featuring a full-screen image gallery, smooth animations, and a contact form. The design emphasizes visual content with minimal distractions, optimized for fast loading times and excellent performance across all devices.

The website includes an admin panel that allows the photographer to easily update their portfolio, manage client inquiries, and showcase their latest work without needing technical knowledge. The clean, minimalist design puts the focus squarely on the stunning photography while providing all necessary information in an intuitive layout.
